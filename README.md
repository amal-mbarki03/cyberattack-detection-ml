# Cyberattack Detection using Machine Learning

## Overview
This project aims to detect cyberattacks using machine learning techniques.

## Features
- Data preprocessing and analysis
- Classification using Random Forest
- Visualization of attack patterns

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Results
The model successfully classifies network traffic as normal or attack with good accuracy.

This project demonstrates a machine learning approach to detect cyberattacks using classification models and data visualization.
